# Les variables à utiliser pour l'espacement dans un formulaire

## Il a été convenu pour l'espacement dans un formulaire :

### Entre les champs :

* Espacement vertical entre 2 champs :

```scss
$form-vertical-gap-mobile: 1rem;
$form-vertical-gap-desktop: 1.5rem;
```

* Espacement horizontal entre 2 champs :

```scss
$form-horizontal-gap: 1.5rem;
```

### Entre les champs et le/les bouton(s) de soumission :

* Espacement vertical entre les champs et le bouton :

```scss
$form-submit-button-vertical-gap-mobile: 3rem;
```

* Espacement horizontal entre les champs et le bouton :
  aujourd'hui, c'est le même espacement qu'entre 2 champs

```scss
$form-horizontal-gap: 1.5rem;
```
