# Media query mixins

La mixin `media`  permettent de gérer facilement les différents breakpoints.

Elle peut prendre les valeurs suivantes en argument : `small`, `medium`, `large`, `xlarge`, `xxlarge`


Elle s'utilise de la manière suivante :
```scss
@use "@styles/utilities";

@include utilities.media(medium) {
    // CSS appliqué aux écrans de taille medium et supérieure
}

@include utilities.media(large) {
    // CSS appliqué aux écrans de taille large et supérieure
}
```


À noter, que la version par défaut doit être la version mobile.

## Exemple

Le SCSS suivant permet de générer un layout qui change en fonction de la taille de l'écran :
```scss
@use "@styles/utilities";

.buttonsContainer {
    display: flex;
    flex-direction: column;

    @include utilities.media(large) {
           flex-direction: row;
    }
}
```
