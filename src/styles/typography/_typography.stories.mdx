# Taille des typographies

Pour utiliser une taille de typographie, on procède de la manière suivante :
```scss
@use "styles/utilities";

.maClasse {
  @extend %text-medium;
}
```
La taille sera alors conforme à la taille médium du Kit UI en desktop et en mobile.
