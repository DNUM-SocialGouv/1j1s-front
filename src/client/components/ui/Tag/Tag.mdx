# Variante cliquable

Le tag a été conçu pour pouvoir être transformé en bouton avec icon, le svg est alors automatiquement centré / aligné avec le texte
<Canvas layout={'centered'}>
    <Tag>
        <button>
            <>Je suis un tag cliquable</>
            <Icon name="close" />
        </button>
    </Tag>
</Canvas>

Pour cela on peut utiliser le code :
```jsx
<Tag>
    <button>
        je suis un tag cliquable
        <Icon name="close" />
    </button>
</Tag>
```
