# Definition

Les ensembles d'onglets / tabs sont construits de plusieurs composants :
<ul>
    <li>`Tabs` qui est l'élément parent et qui peut prendre les [attributs d'une `div` HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) (un composant par ensemble d'onglets) </li>
    <li>`TabsLabel` qui est l'élément parent des `Tab` et englobe donc les entêtes, et qui peut prendre les [attributs d'une `div` HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) (un composant par ensemble d'onglets) </li>
    <li>`Tab` qui représente l'entête de l'onglet et et qui peut prendre les [attributs d'un `button` HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) (un composant `Tab` par onglet)</li>
    <li>`TabPanel` qui représente le contenu d'un onglet et qui peut prendre les [attributs d'une `div` HTML](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) (un composant par ensemble d'onglets) </li>
</ul>

## Example
Ainsi le code suivant :
```typescript jsx
<Tabs>
    <TabsLabel>
        <Tab> Onglet 1 </Tab>
        <Tab> Onglet 2 </Tab>
        <Tab> Onglet 3 </Tab>
    </TabsLabel>
    <TabPanel> Contenu de l'onglet 1 </TabPanel>
    <TabPanel> Contenu de l'onglet 2 </TabPanel>
    <TabPanel> Contenu de l'onglet 3 </TabPanel>
</Tabs>
```
donne :
<Canvas>
    <Tabs>
        <TabsLabel>
            <Tab> Onglet 1 </Tab>
            <Tab> Onglet 2 </Tab>
            <Tab> Onglet 3 </Tab>
        </TabsLabel>
        <TabPanel> Contenu de l'onglet 1 </TabPanel>
        <TabPanel> Contenu de l'onglet 2 </TabPanel>
        <TabPanel> Contenu de l'onglet 3 </TabPanel>
    </Tabs>

</Canvas>
