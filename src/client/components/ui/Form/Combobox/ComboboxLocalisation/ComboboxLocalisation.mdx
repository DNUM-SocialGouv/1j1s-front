# Definition

`ComboboxLocalisation` est le composant utilisé dans les formulaires de recherche.

Il permet de renseigner une localisation par rapport à la recherche effectuée.

## Props

### ComboboxLocalisation

Le `ComboboxLocalisation` peut prendre les props suivantes :

<Controls />

**Mais également tous [les attributs d'un `input` HTML natif](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input)**

Pour construire la valeur par défaut, utiliser la fonction `createLocalisationDefaultValueFromQuery` exporté dans le fichier du `ComboboxLocalisation`.

## Informations supplémentaires

### Gestion des états
Le `ComboboxLocalisation` embarque une gestion interne des états :
- affichage du chargement pendant le chargement des localisations
- affichage des résultats correspondants à l'input utilisateur
- affichage en erreur lorsque le chargement des localisations est en erreur

### Source du référentiel de `ComboboxLocalisation`
Aujourd'hui, le référentiel des localisations est fourni par LocalisationService.
