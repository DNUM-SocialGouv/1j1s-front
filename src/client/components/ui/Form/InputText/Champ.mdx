# Definition

Le composant `Champ` permet de lier automatiquement le label, le champ, l'aide à la saisie et le message d'erreur.

<Primary />

Il gère également automatiquement l'affichage du message d'erreur custom ainsi que son contenu.

<Canvas of={ChampStories.Required} />

## Exemple

```tsx
<Champ>
	<Champ.Label>Pays <Champ.Label.Complement>(requis)</Champ.Label.Complement></Champ.Label>
	<Champ.Input render={Combobox} aria-label="Pays" required={true}>
		<Combobox.Option>France</Combobox.Option>
		<Combobox.Option>Allemagne</Combobox.Option>
		<Combobox.Option>Suisse</Combobox.Option>
		<Combobox.Option>Norvège</Combobox.Option>
		<Combobox.Option>Espagne</Combobox.Option>
		<Combobox.Option>Italie</Combobox.Option>
	</Champ.Input>
	<Champ.Error/>
</Champ>
```

Le composant `<Champ.Input>` prend en props un composant à utiliser (e.g. `Input`, `Combobox`, `Textarea`, ...) par la props `render`.
Remarque : si le composant à utiliser prend des `children`, ces `children` peuvent être donnés au `<Champ.Input>`comme dans l'exemple ci-dessus dans lequel `<Combobox>` prend des `<Combobox.Option>` en `children`

**Le composant à afficher doit au moins accepter les props suivantes :**
```ts
type ComponentChildrenPropsNecessary = {
	onChange?: (event: React.ChangeEvent<HTMLInputElement>, ...args: any[]) => void;
	onTouch?: (touched: boolean, ...args: any[]) => void
	ref?: React.Ref<HTMLInputElement>
	'aria-describedby'?: string
	id?: string
}
```
