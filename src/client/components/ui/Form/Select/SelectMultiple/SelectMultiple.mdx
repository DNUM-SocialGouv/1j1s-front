# Select Multiple

## Definition

Le composant permet de sélectionner un sous-ensemble d'éléments parmi une liste fixe. Cette implémentation vise à reproduire le comportement d'un `select` natif combiné avec son attribut `multiple`, avec une interface plus proche du select simple. On pourra aussi se référer au pattern ARIA [select-only combobox](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/examples/combobox-select-only/).

Ce composant expose un sous-composant `SelectMultiple.Option` pour définir la liste des choix.

<Primary/>

À la différence du [Combobox](/docs/components-form-combobox--docs), l'utilisateur est contraint de choisir une option dans la liste et ne peut pas renseigner un champ à la main.
